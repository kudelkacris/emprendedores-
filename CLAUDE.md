# EMPRENDEDORES — Landing Pages Santa Clara del Mar

Template reutilizable para micro-emprendedores locales.

## Modelo
- $50–80 USD pago único (construcción)
- $5 USD/mes (hosting en Netlify)
- Dominio opcional: cliente compra en NIC.ar

## Stack
HTML puro · CSS vanilla · Mobile-first · Sin dependencias · Netlify

## Archivos
- `template/index.html` — template base editable
- `clientes/` — una carpeta por cliente con su versión final
